Bienvenidos a mi primer proyecto personal. Un gestor para clubes deportivos con las siguientes opciones:

Generar nuevos clubes
Agregar, modificar o borrar empleados de la Base de Datos
Gestionar turnos filtrando por deporte, superficie, fecha y hora
Cambiar datos del club como dirección, telefono, etc.
English

This is my first personal project. A program to manage Sport Clubs, with some options like:

Create new Clubs
Add, edit or remove employers in Data Base
Search for avaible hours to practice sports (You can filter by date, sport, surface and time )
Change information about the club like adress, phone...


Instrucciones para usar

El programa por defecto tiene el club "Todo Deportes" al que se puede ingresar con las credenciales
User: admin
Pass: admin

Si lo desea, ud puede generar un nuevo club donde se creara un usuario con las mismas credenciales anteriores.
